# l2-patcher
AHKv2 gui utils that apply human-readable (decoded) changes to .txt system files for C1 client

## Что делает
Программа находит все .txt файлы в патче из папки system, копирует из папки с игрой в папку "orig" аналогичные файлы (таким образом всегда (если вы не забыли сделать full check) работает с актуальными игровыми файлами и позволят снимать галки в лоунчере, без отката на устаревшие файлы).  
Бекапит текущий набор патченных файлов  
Раскодирует игровые файлы  
Вносит те изменения, что вы указжете в файлах "resources"  
Кодирует обратно согласно протоколу 211 (C1 клиент)  
Заменяет файлы в папке "files" новыми  


## Инструкция
1. Поместите папку `scripts` в ваш патч (рядом с папками `files` и `orig`):
   ```text
   Lineage2/
   └── Patches/
       └── Your_patch/
           ├── files/
           ├── orig/
           └── scripts/
   ```
2. Запустите **full check**.

3. Запустите **patcher.ahk** -> отобразится список всех .txt файлов из вашего патча  
4. Раздавите муху и дождитесь завершения работы скрипта (алгоритмы оптимизации поиска не применялись, поэтому процесс занимает значительное время)  

Если статус "ошибка", то посмотрите подробное описание в графе состояния или в появившемся MsgBox и исправьте проблему  
Если статус "файл не изменен", то скорее всего вы или не выполнили full check или ваши изменения не к чему применить   
Если статус "файл в патче уже актуален", то это значит, что текущая версия файла в патче уже совместима с игрой  
Если статусы "готово" или "файл в патче актуален", то все пропатчено, можно в лоунчере выставлять галку и запускать игру.  

## Как создавать свои изменения
txt файл UTF-8-BOM  
id обязательный параметр  
остальные поля через tab в таком же формате, что в оригинальном файле игры. порядок данных в строках и самих строк не имеет значения.  

если требуется применить дополнительные изменения к уже имеющимся - поместите файлы (с такими же названиями) в папку step2 (step3, 4 и тд)
